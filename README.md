# E-Approval

전자결재 시스템을 주제로 한 백엔드 중심 프로젝트<br/>
실무에서 경험한 전자결재 도메인을 바탕으로, 설계부터 구현, 테스트, 배포까지 직접 진행하는 것을 목표로 함

---

## 1. 프로젝트 개요
- 주제: 전자결재 시스템
- 목표:
    - 결재/문서/결재선 도메인 설계
    - REST API 설계 및 구현
    - 인증/인가(JWT) 적용(Session/JWT)
    - Redis, Docker, MySQL 활용
    - 테스트 코드 작성

---

## 2. 기술 스택
